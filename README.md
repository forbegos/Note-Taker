# Note Taker APP

This application allows you to write and save notes as well as delete notes you don't need anymore. Notes are saved and retrieved from a json file.

## Technologies Used

<ol>
    <li>HTML</li>
    <li>CSS</li>
    <li>Javascript</li>
    <li>Node.js</li>
    <li>Express.js</li>
</ol>
